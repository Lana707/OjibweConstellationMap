# Ojibwe Constellation Map 
A map including constellations and descriptions for some Ojibwe constellations. 
Please note that there are many interpretations and variations for these traditional stories and constellations, 
those represented are not intended to represent all Ojibwe communities or any Indigenous beliefs. All resources and references to material are included.

Names for stars excluding the North Star use iau names. 
## Created using d3-celestial https://github.com/ofrohn/d3-celestial 
Constellations are mapped using GeoJSON and coordinates of RA/Dec converted to longitude and latitude
